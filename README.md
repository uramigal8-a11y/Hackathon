# Документація проекту

## Огляд проекту
Цей проект присвячений аналізу телеметрії дронів, що дозволяє отримувати, обробляти та візуалізувати дані, отримані з дронів. Використання системи аналізу телеметрії допоможе поліпшити ефективність і безпеку польотів, а також надасть користувачам можливість більш детально розуміти поведінку дронів під час роботи.

## Обґрунтування вибору технологій
У цьому проекті були обрані мови програмування C++ та Python. Зокрема, їх використання обумовлено наступними причин:

1. **C++**: Ця мова програмування є сильно типізованою та високо продуктивною. Вона надає можливість управління пам'яттю, що є критично важливим для обробки великих обсягів даних, отриманих з дронів. Системи, реалізовані на C++, можуть працювати швидше, що є суттєвим для задач реального часу, таких як обробка телеметрії.

2. **Python**: Ця мова є гнучкою та має величезну кількість бібліотек для обробки даних, статистики та візуалізації, що робить її ідеальною для аналізу даних. Python дозволяє швидко прототипувати рішення та зосередитись на алгоритмах і аналітиці, не витрачаючи багато часу на реалізацію низькорівневих деталей.

Використання обох мов програмування дає можливість поєднати швидкість ��иконання коду (C++) з гнучкістю та простотою розробки (Python).

## Інсталяція
1. Склонуйте репозиторій:
   ```bash
   git clone https://github.com/uramigal8-a11y/Hackathon.git
   ```
2. Перейдіть в теку:
   ```bash
   cd Hackathon
   ```
3. Встановіть необхідні залежності:
   Для C++ проекту вам можуть знадобитися такі бібліотеки, наприклад, OpenCV, а для Python – NumPy, Pandas та Matplotlib, які можна встановити за допомогою pip:
   ```bash
   pip install -r requirements.txt
   ```

## Використання
Запустіть проект за допомогою:
```bash
python main.py
```

## Структура проекту
- `main.py`: Головний файл для запуску проекту.
- `src/`: Папка, що містить C++ код.
- `data/`: Датафрейми та зібрані дані.
- `requirements.txt`: Список Python бібліотек, необхідних для запуску проекту.
